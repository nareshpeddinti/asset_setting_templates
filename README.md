# Asset Types Management (V0 → Cursor)

V0 prototype for asset types, fieldsets, and bulk create — now in Cursor.

## Run locally

```bash
npm install   # already done
npm run dev   # http://localhost:3000
```

## Build

```bash
npm run build
npm start
```

## Project skill

The **asset-config** skill in `.cursor/skills/asset-config/` applies when you work on asset hierarchy and fieldset logic. Use it by asking to "create asset hierarchy," "define fieldsets," or "map these attributes" in this repo.
